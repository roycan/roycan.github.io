COMMENTS SECTION

This section contains the collated comments from the IC personnel who reviewed the website screens. These were gathered during a series of walkthroughs and user acceptance tests conducted at the Commission. Note that major comments were immediately implemented, and are reflected in the screens shown in the previous section. Other comments will be revisited as the system is developed.

  • Change the input format for the schedules and summary; align the design with the format of the output
  • Create an attachment column on companies' side for supporting documents/uploading of excel

                Examiner’s side:

  • Create an organizational structure for quarterly submissions. See Annual
  • Remarks column on summary
